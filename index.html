Como adicionar ao Notion

  1. Personalize o widget usando os controles acima ou escolha um estilo predefinido
  2. Clique em "Copiar Link" quando estiver satisfeito com o resultado
  3. No Notion, digite /embed e pressione Enter
  4. Cole o link copiado no campo de URL
  5. Ajuste o tamanho do bloco conforme necessário
Dica: Para melhor resultado, configure a altura do embed para aproximadamente 150-200 pixels.
